# Weakly_Supervised_Human_Matting-PengyueZhang
Weakly supervised human matting based on point annotations

## Abstract
This project implements a weakly supervised human matting method using only sparse point annotations,
including pseudo-label generation, label cleaning, and a ResNet50 + attention-enhanced model.

## Environment
- Python 3.8+
- PyTorch 1.10+
- CUDA 11.0+ (optional)
- OpenCV, PIL, NumPy, Matplotlib
